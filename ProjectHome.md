## Simple Java FTP Server ##
I've implemented an Active and Passive FTP Server using Java as college assignment for the network course.
It doesn't support all commands like Extended Passive or whatever...just the basic FTP commands...commands like LIST,RETRE,STOR,CWD,PWD,USER,PASS,..etc
